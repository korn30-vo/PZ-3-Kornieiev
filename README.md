Цей проєкт містить реалізацію алгоритму кластеризації k-means для аналізу даних у Python. У роботі розглянуто як ручну реалізацію алгоритму, так і використання бібліотеки sklearn з методом k-means++ та методом ліктя для визначення оптимальної кількості кластерів.

Проєкт складається з двох частин: у першій реалізовано алгоритм k-means для набору даних про кредити клієнтів, у другій — виконано кластеризацію даних оптових клієнтів із попередньою стандартизацією ознак.
